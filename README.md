# NovelTrend

Light novel & anime rankings powered by RanobeDB and AniList.

## Deploy to GitHub Pages

1. Go to your repo → **Settings → Pages**
2. Under **Source**, select **GitHub Actions** → Save
3. Push any commit to `main` — it auto-deploys

Live at: `https://jayroring2013.github.io/bxh/`

## Run locally

```bash
npm install
npm run dev
```
